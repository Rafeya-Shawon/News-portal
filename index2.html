let is block-scoped,var is function scoped. let does not allow to redeclare variables,var allows to redeclare variables. Hoisting does not occur in let, Hoisting occurs in var.

  • Syntactical differences
  • this value (execution context)
  • Usage as methods
  • Usage as constructors
  • arguments binding
